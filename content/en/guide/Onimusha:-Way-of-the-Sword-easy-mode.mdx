<Section icon={Swords} title="Key Features of Easy Mode vs. Action Mode">

These changes are designed to streamline gameplay and reduce frustration, allowing players to focus on the game's narrative and exploration. Below is a comparison of how these two modes differ.

| Feature             | Story (Easy) Mode                                    | Action (Normal) Mode                                 |
| :------------------ | :--------------------------------------------------- | :--------------------------------------------------- |
| **Enemy Strength**  | Significantly decreased attack power and defense     | Standard enemy attack power and defense              |
| **UI Prompts**      | More obvious on-screen button prompts for actions    | Standard UI with less explicit guidance              |
| **Input Timings**   | More forgiving input timings for special moves       | Strict input timings for deflecting and Issen        |
| **Deflect/Issen**   | Easier execution for deflecting and Issen            | Requires precise timing and skill for deflecting/Issen |
| **Overall Challenge** | Incredibly easy, little challenge for typical grunts | Moderate challenge, requires strategic combat        |
| **Boss Fights**     | Bosses are significantly easier to defeat (e.g., Ganryu in 1-2 minutes) | Bosses require multiple attempts and careful strategy |
