### 清水寺での敵との遭遇

寺院には様々な幻魔がうろついています。標準的な剣士、素早い悪魔、そしてより大きく、より頑丈な生き物と対峙することになります。それぞれの敵タイプに合わせて戦闘スタイルを適応させましょう。

| 敵の種類       | 攻撃パターン            | 弱点 / 戦略             |
| :--------------- | :---------------------- | :---------------------- |

| **俊敏な悪魔**   | 素早い突進、空中攻撃    | 回避後、素早い攻撃で反撃 |
| **ブルート幻魔** | 遅い、重い振り回し      | 弾きに集中し、足を狙ってよろめかせる |
| **蜘蛛の魔物**   | 網の罠、遠距離攻撃      | 素早く接近し、詠唱を中断させる |

### 蜘蛛の魔物への対処

デモの内容で言及されている注目すべき遭遇の一つに「隠れている忌まわしき蜘蛛」があります。これは、特定の戦術を必要とするミニボス、あるいは特に厄介な敵を示唆しています。

| 蜘蛛の魔物の戦術 | 説明                                       | 対策戦略                               |
| :------------------ | :----------------------------------------- | :------------------------------------- |
| **網の罠**          | 武蔵の動きを遅らせ、無防備にする。         | ドッジロールまたは素早い攻撃で脱出する。 |
| **遠距離投射物**    | 腐食性の液体や小さな投射物を吐き出す。     | 距離を保ち、飛んでくる攻撃をパリィする。 |
| **近接攻撃**        | 近距離攻撃、武蔵を吹き飛ばすことがある。   | 弾きまたは回避後、脚に反撃する。       |
| **手下の召喚**      | 小さな蜘蛛を呼び出し、武蔵を圧倒する。     | まず手下を優先するか、範囲魔法を使用する。 |

<Callout type="success" title="難易度設定">
『鬼武者 Way of the Sword』には「ストーリー」と「アクション」の難易度があります。アクションゲーム初心者の方は「ストーリー」でより遊びやすい体験ができ、「アクション」は真の挑戦を求めるベテランにおすすめです。ゲームはすべてのスキルレベルで楽しめることを目指しています。
</Callout>

<Section icon={Trophy} title="ボス戦：佐々木巌流">
デモのクライマックスは、佐々木巌流との対決です。彼は自身の鬼の籠手を振るう手ごわいライバルです。この戦いは、防御的な戦闘とタイミングの習熟度を試す真の試練となるように設計されています。

佐々木は武蔵と同様に、魂を集めた後に強力な攻撃を繰り出すことができます。彼の戦闘スタイルは武蔵のそれに似ている可能性が高く、この戦いはスキルと正確さの決闘となるでしょう。

| 佐々木巌流の能力 | 説明                                     | 対策戦略                               |
| :---------------------- | :--------------------------------------- | :------------------------------------- |
| **鬼の籠手攻撃**        | 強力な、魂を込めた攻撃。                 | スタミナを削るために**パリィ/弾きが不可欠**。 |
| **高速コンボ**          | 素早い剣術の連続技、中断しにくい。       | ガードし、最後の攻撃後に隙を探す。     |
| **一閃カウンター**      | 攻撃をミスすると、カウンターを仕掛けることがある。 | 忍耐強く、隙が明確なときだけ攻撃する。 |
| **魂吸収**              | より強力な攻撃のためにエネルギーを集める。 | 可能であればこのアニメーションを中断するか、大ダメージに備える。 |

<Callout type="tip" title="忍耐と観察">
この戦いを急いではいけません。佐々木のパターンを観察し、彼が無防備になる瞬間を見極め、タイミングの良いパリィと一閃カウンターでそこを突きましょう。ここでは、あなたの防御オプションが最高の攻撃手段となります。
</Callout>
