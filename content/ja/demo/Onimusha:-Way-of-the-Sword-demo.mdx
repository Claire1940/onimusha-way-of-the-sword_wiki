多くのアクションゲームとは異なり、『鬼武者 Way of the Sword』は防御的なプレイスタイルを推奨していません。過度にガードやパリィを試みるよりも、全力で攻め立てる方が効果的であることが多いです。戦闘システムには独自のスタミナ要素があり、敵に攻撃を当てると黄色のスタミナバーが減少します。これを使い果たした敵は怯んだり、致命的な攻撃を叩き込む隙を見せたりします。そのため、常に敵の攻撃範囲内に留まり、執拗に攻撃してガードを崩す立ち回りが求められます。

### 戦闘メカニクスの概要

| メカニクス | 説明 | ゲームプレイへの影響 |
| :------------ | :-------------------------------------------------------------------------- | :----------------------------------------------------- |
| **アグレッション**| 攻撃的な動きと素早い撃破を優先する。 | 被ダメージを抑え、敵をより速く排除できる。 |
| **パリィシステム**| 敵の攻撃を弾き返す正確なタイミング。 | 生存に不可欠だが、主要な防御手段ではない。 |
| **スタミナバー** | 敵の黄色いバーで、攻撃を当てると減少する。 | スタミナを削りきると、大きな隙やチャンスが生まれる。 |
| **ブレイクシステム**| プレイヤーもダメージを受けるとスタミナが低下し、即死につながる。 | 戦略的な位置取りとターゲットの優先順位付けを促す。 |
| **回復** | ポーションが使用可能だが、戦闘中の使用は動作が重い。 | 状況に応じた回復が必要で、慎重なタイミングが求められる。 |

<Section icon={Target} title="武蔵の旅：ストーリーとキャラクターの印象">
プレイヤーは伝説の侍、宮本武蔵として、異世界の古都・京都に足を踏み入れます。この世界の武蔵は、半魔半人の存在である「鬼武者」になることを余儀なくされています。彼は手に入れた異能を正義のために使い、さまざまな種類の魂を吸収することで、スタミナ、体力、そして奥義エネルギーを強化していきます。体験版では、喋る籠手や、かつての修行仲間（彼もまた鬼武者である可能性がある）との対立を予感させる、深みのある物語が示唆されています。

### 魂の吸収メカニクス

| 魂の種類 | 効果 | 戦闘での活用 |
| :------------ | :----------------------------------------- | :----------------------------------------------------- |
| **スタミナの魂** | 最大スタミナを増加させる。 | より攻撃的なコンボや機敏な動きが可能になる。 |
| **体力の魂** | 体力を回復、または最大値を増加させる。 | 過酷な遭遇戦での生存率を高めるために不可欠。 |
| **奥義の魂** | 強力なアルティメットアビリティの燃料となる。 | 強敵に対して壊滅的な一撃を放つことができる。 |
| **環境の魂** | 壺などのオブジェクトを破壊することで入手。 | 軽微なバフやリソースを提供する。 |

体験版のストーリー要素は最小限に留められており、挑戦的なゲームプレイの確立に重点が置かれています。しかし、喋る籠手とのやり取りや、「善の鬼武者対悪の鬼武者」の対立を予感させるカットシーンの存在は、豊かな物語の可能性を感じさせます。籠手によって授けられる「オン・ビジョン（魂の視覚）」能力を使えば、新しい道を切り開くスピリチュアルな結び目を見つけることができ、探索に軽いバックトラッキングの要素を加えています。

<Callout type="tip" title="探索のヒント">
武蔵の「オン・ビジョン」を活用して、隠された道や秘密を見つけ出しましょう。単純な戻り道に見えるかもしれませんが、これらの寄り道は貴重なリソースやショートカットにつながることが多いです。
</Callout>

<Section icon={Shield} title="敵のバリエーションと難易度の推移">
体験版では、敵タイプの満足のいく進行が見て取れます。序盤、プレイヤーは比較的弱い人間型の敵（いわば剣を持ったゾンビ）に遭遇します。しかし、開始から15分も経たないうちに、装甲を纏ったトカゲのようなクリーチャーや、重装甲で剣を振るう巨漢が登場し、難易度が急上昇します。これらの強敵は複数の体力ゲージを持ち、怯ませるのが非常に困難なため、より精密な戦闘と戦略的な魂の吸収が求められます。

### 遭遇する敵の種類

| 敵の種類 | 特徴 | 戦闘戦略 |
| :--------------- | :------------------------------------------------------ | :----------------------------------------------------- |
| **下級兵** | 人間型、低体力、怯みやすい。 | 素早いコンボで倒し、魂稼ぎに利用する。 |
| **装甲リザード** | 中程度の体力、速い攻撃、手数が求められる。 | ガードを崩すか、隙を見つけることに集中する。 |
| **重装甲兵** | 高体力、複数ゲージ、怯みにくい。 | 奥義、パリィ、攻撃的なコンボを駆使する。 |
| **狙撃兵** | 遠距離攻撃、高所に配置されることが多い。 | 妨害を避けるため、最優先で排除する。 |

本作の難易度は非常にやりがいがあり、すべての死が学習体験となります。死亡後に即座に戦闘をリスタートできる機能（体験版特有の可能性もあります）は、継続的な練習と戦闘スキルの洗練を促します。この設計は、スキル重視の攻撃的な戦闘というゲーム全体のコンセプトと合致しています。

</Callout>
