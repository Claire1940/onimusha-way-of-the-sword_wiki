<Section icon={Swords} title="戦闘メカニクスと侍の道">

今作の戦闘は、近年の「ソウルライク」のトレンドとは一線を画し、精度、パリィ、そして環境認識に焦点を当てています。武蔵の主な武器は刀ですが、システムの奥深さは敵のスタミナや位置取りとの相互作用にあります。特に注目すべきは「ステアリング（誘導）」メカニクスです。パリィを維持することで、武蔵は相手の勢いを物理的に受け流し、環境トラップに追い込んだり体勢を崩させたりすることができます。

| 戦闘アクション | メカニクスタイプ | プレイヤーのメリット |
| :--- | :--- | :--- |
| **通常攻撃** | 攻撃的 | 基本ダメージを与え、敵のスタミナを削る。 |
| **長押しパリィ** | 防御的/戦術的 | 敵を環境トラップの方へ「ステアリング（誘導）」できる。 |
| **ガード構え** | 防御的 | スタミナを消費して全方向からの攻撃を防ぐ。 |
| **崩し一閃** | 処刑 | 通常の敵を部位欠損と共に即座に倒す。 |
| **戦闘バフ** | パッシブ | パリィや回避を連続で成功させた後に付与される。 |

環境は単なる背景ではありません。テーブルをひっくり返して一時的な遮蔽物を作ったり、京都の狭い路地を利用して超自然的な脅威を一本道に誘い込んだりすることができます。この戦術的なレイヤーは、独自のパリィやカウンターを駆使してくるライバルの剣客たちと対峙する際に不可欠となります。

<Callout type="warning" title="スタミナ管理">
ガード構えに頼りすぎると、すぐにスタミナが枯渇してしまいます。ボス戦でスタミナ切れを起こすと、武蔵はガード不能な「鬼」の攻撃に対して無防備になり、一撃で戦死するリスクがあります。
</Callout>

</Section>

<Section icon={Sparkles} title="鬼の籠手と魂の吸収">

成長システムの中心となるのは、武蔵が装着している意思を持つ籠手です。このアーティファクトは倒した幻魔の魂を吸収し、それらは3つの色に分類されます。これらのリソースを管理することが、一本道のダンジョンを生き抜くか、江戸時代の超自然的な勢力に圧倒されるかの分かれ道となります。

<CardGrid cols={3}>
  <Card title="黄魂">
    - **機能**: 体力回復
    - **入手先**: 敵のドロップ、または「鬼の兵装」で敵を攻撃した際に発生。
    - **優先度**: ボス戦において極めて高い。
  </Card>
  <Card title="赤魂">
    - **機能**: 強化用の通貨
    - **入手先**: すべての敵からの標準的なドロップ。
    - **優先度**: 長期的な戦力強化に不可欠。
  </Card>
  <Card title="青魂">
    - **機能**: 鬼エネルギー
    - **入手先**: レアドロップ、または特定の環境ギミック。
    - **優先度**: 強力な「鬼の兵装」を発動するために必要。
  </Card>
</CardGrid>

「鬼の眼（Oni Vision）」能力は、籠手の有用性をさらに高めます。このモードを起動することで、武蔵は幻魔の幻術を見破り、群衆に紛れた敵の正体を暴いたり、破壊可能な壁の向こうに隠された赤魂の宝箱を見つけたりすることができます。

| 籠手の機能 | 使用リソース | 戦闘での活用 |
| :--- | :--- | :--- |
| **鬼の兵装** | 青魂 | 強力なダメージを与えるスペクトル武器を召喚する。 |
| **鬼の眼** | パッシブ/アクティブ | 隠れた幻魔や秘密のアイテムを検知する。 |
| **魂吸収** | 自動/手動 | 撃破後、離れた場所から魂を引き寄せる。 |
| **強化システム** | 赤魂 | 刀の鋭利さや籠手の容量を強化する。 |

<Callout type="success" title="ボス攻略のヒント">
ボスに対して「崩し一閃」を決めた際、2つの選択肢が与えられます。「即座に大ダメージを与える」か、「大量の魂を抽出する」かです。体力が少ない場合は、安全策として魂の抽出を選び、黄魂を確保するのが賢明です。
</Callout>

</Section>

<Section icon={Shield} title="次世代ハードウェア向けの技術分析">

**『鬼武者 Way of the Sword』Switch 2版** の可能性は、任天堂の次世代機のスペックにかかっています。2026年6月の業界レポートに基づくと、新ハードウェアはDLSSのようなアップスケーリング技術をサポートしており、これは本作の緻密なキャラクターモデルと滑らかな 60 FPS の戦闘アニメーションを維持するために不可欠です。

<Steps>
  <Step num="1" title="エンジンの互換性">
    カプコンの RE Engine は非常にモジュール化されています。『Way of the Sword』で使用されているバージョンには部位欠損システムのための高度な物理演算が含まれており、これには相応のCPU負荷がかかります。
  </Step>
  <Step num="2" title="メモリ要件">
    本作は、京都の街並みから寺院の境内へとシームレスに遷移します。Switch 2で噂されている増強されたRAMは、ローディングによるカクつきを防ぐために必要となるでしょう。
  </Step>
  <Step num="3" title="入力遅延">
    一閃のメカニクスはフレーム単位のパリィに依存しているため、携帯・据置ハイブリッド機への移植では、戦闘の流れを損なう「入力遅延」を避けるために安定したフレームレートを維持することが絶対条件となります。
  </Step>
</Steps>
